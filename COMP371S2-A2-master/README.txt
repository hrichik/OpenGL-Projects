Program Controls:

  Model selection controls:
  
    - '1' -> Select heart model (Jennifer's model)
    - '2' -> Select star model (Anna's model)
    - '3' -> Select third model (Hrichick's model)
    - '4' -> Select fourth model (Matthew's model)

  Model size controls:
    
    - 'U' -> Scale-up
    - 'J' -> Scale-down
    
  Model position & orientation controls:
  
    - 'A' -> Move left
    - 'D' -> Move right
    - 'W' -> Move up
    - 'S' -> Move down
    
    - 'I' -> Rotate left 5 degrees about Y axis
    - 'O' -> Rotate right 5 degrees about Y axis
    
    - 'Z' -> Small move back on Z axis
    - 'X' -> Continuous move back on Z axis
    - 'C' -> Small move forward on Z axis
    - 'V' -> Continuous move forward on Z axis
    
    - 'T' -> Rotate clockwise on X axis
    - 'G' -> Rotate counter-clockwise on X axis
    - 'H' -> Rotate clockwise on Y axis
    - 'F' -> Rotate counter-clockwise on Y axis
    - 'Y' -> Rotate clockwise on Z axis
    - 'R' -> Rotate counter-clockwise on Z axis
    
    - 'P' -> Points
    - 'L' -> Lines
    - 'T' -> Triangles
  
  World orientation controls:
    
    - 'Left arrow' -> Anti-clockwise rotation about x axis
    - 'Right arrow' -> Anti-clockwise rotation about negative x axis
    - 'Up arrow' -> Anti-clockwise rotation about y axis
    - 'Down arrow' -> Anti-clockwise rotation about negative y axis
    
  Model Rendering Mode controls:
  
    - 'P' -> Points
    - 'L' -> Lines
    - 'T' -> Triangles
    
  Camera Controls:
  
    - 'Mouse right button pressed' + 'Mouse movement in x axis' -> Pan
    - 'Mouse middle button pressed' + 'Mouse movement in y axis' -> Tilt
    - 'Mouse left button pressed' + 'Mouse movement' -> Zoom in/out of the scene
